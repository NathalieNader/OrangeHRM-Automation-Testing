# OrangeHRM Test Automation Framework

![Java](https://img.shields.io/badge/Java-24-orange)
![Selenium](https://img.shields.io/badge/Selenium-4.21-green)
![TestNG](https://img.shields.io/badge/TestNG-7.10-blue)
![Allure](https://img.shields.io/badge/Allure-Reporting-purple)

UI automation testing framework for OrangeHRM developed using Selenium WebDriver, Java, TestNG, Page Object Model (POM), Fluent Pattern, and Allure Reporting.

---

# Project Overview

## Application Under Test

OrangeHRM Demo Application

## Scope

- Login Functionality
- Dashboard Validation
- PIM Module
- Employee Search
- Menu Navigation
- UI Validation
- Error Handling
